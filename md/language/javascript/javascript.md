# JavaScript

## 実行保法

|              内容              |                      記載サンプル                       |               備考               |
| ------------------------------ | ------------------------------------------------------- | -------------------------------- |
| HTML直書き(scriptタグ内に記載) | `<script>alert("XXX")</script>`                         | headタグ or bodyタグ内に記載する |
| JSファイル読み込み             | `<script type="text/javascript" src="xxx.js"></script>` | (同上)                           |

※HTML読み込みを優先させるため、scriptタグはbodyタグ内の最後に記載するのが望ましい。
