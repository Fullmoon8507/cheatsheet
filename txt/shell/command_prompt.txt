【ファイル・フォルダの一覧情報】
|----------+------------+----------------------------------------------------------------|
| コマンド | オプション | 内容                                                           |
|----------+------------+----------------------------------------------------------------|
| dir      | -          | カーレントディレクトリのファイル名とフォルダ名をリスト表示     |
|----------+------------+----------------------------------------------------------------|
|          | /A:属性    | 指定した属性のファイルのみ表示                                 |
|          |            | 　D:ディレクトリ                                               |
|          |            | 　R: 読み取り専用                                              |
|          |            | 　H: 隠しファイル                                              |
|          |            | 　A: アーカイブ                                                |
|          |            | 　-: その属性以外                                              |
|----------+------------+----------------------------------------------------------------|
|          | /b         | ファイル名またはディレクトリ名のみ表示                         |
|          | /s         | サブディレクトリも表示する                                     |
|----------+------------+----------------------------------------------------------------|
| tree     | -          | カーレントディレクトリ以下のディレクトリ構造をツリー形式で表示 |
|----------+------------+----------------------------------------------------------------|
|          | /f         | ファイル名も含めてツリー形式で表示                             |
|          | /a         | ツリー構造をASCII文字で表示                                    |
|----------+------------+----------------------------------------------------------------|
