【標準出力】
|----------+--------+----------------------------|
| 分類     | 言語   | 文法                       |
|----------+--------+----------------------------|
| 改行あり | C      | printf("xxx\n");           |
|          | C++    | cout << "xxx" << endl;     |
|          | Java   | System.out.println("xxx"); |
|          | Python | print("xxx")               |
|----------+--------+----------------------------|
| 改行なし | C      | printf("xxx");             |
|          | C++    | cout << "xxx";             |
|          | Java   | System.out.print("xxx");   |
|          | Python | print("xxx")               |
|----------+--------+----------------------------|

【標準入力】
|--------+------------------------------------------+---------------------------------------------------------------------------------------|
| 言語   | 文法                                     | 備考                                                                                  |
|--------+------------------------------------------+---------------------------------------------------------------------------------------|
| C      | int (変数) = getchar();                  | EOFまで一文字ずつ取得。                                                               |
|        | gets(char *s);                           | EOFまで一行ずつ取得。                                                                 |
|        | fgets(char *row, int len, FILE *fp);     | fpからlen文字分取得し、文字列の先頭ポインタを返す。コンソール場合は*fpにstdinを指定。 |
|        | scanf("(変換指定子)", %(変数));          | 変換指定子で指定した形式で取得。                                                      |
|--------+------------------------------------------+---------------------------------------------------------------------------------------|
| C++    | cin >> (変数);                           |                                                                                       |
| Java   | Scanner (変数) = new Scanner(System.in); |                                                                                       |
| Python | (変数) = input()                         |                                                                                       |
|--------+------------------------------------------+---------------------------------------------------------------------------------------|
