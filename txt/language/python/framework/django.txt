【プロジェクト作成】

| コマンド                                   | オプション | 内容                     |

|---------------------+-------------------------+-------------------------------|
| コマンド            | オプション              | 内容                          |

|---------------------+-------------------------+-------------------------------|

|------------------------------------------------+------------+------------------------|

****************************************************************************************************
* Model（データベース）関連
****************************************************************************************************
【マイグレーション関連】
|----------------------------------------------------------------------+------------+--------------------------------------------------------|
| コマンド                                                             | オプション | 内容                                                   |
|----------------------------------------------------------------------+------------+--------------------------------------------------------|
| python manage.py makemigrations                                      | -          | マイグレーションファイルの作成                         |
| python manage.py migrate                                             | -          | マイグレーション                                       |
|----------------------------------------------------------------------+------------+--------------------------------------------------------|
| python manage.py showmigrations                                      | -          | マイグレーションの状態確認                             |
| python manage.py showmigrations [アプリケーション名]                 | -          | 指定したアプリケーション名のマイグレーションの状態確認 |
|----------------------------------------------------------------------+------------+--------------------------------------------------------|
| python manage.py migrate [アプリケーション名] [マイグレーション番号] | -          | マイグレーションのロールバック                         |
|----------------------------------------------------------------------+------------+--------------------------------------------------------|
| python manage.py migrate [アプリケーション名] zero                   | -          | マイグレーションの初期化（drop table）                 |
|----------------------------------------------------------------------+------------+--------------------------------------------------------|


【CRUDの基本】
- モデルクラス.モデルマネージャ.メソッド()
	- モデルクラス：django.db.models.Model継承クラス。
	- モデルマネージャ：デフォルトはobjectsという名前。
	- メソッド：SQLを実行する各メソッド。

【データ取得】
|-----------------------------------+----------------------------|
| 構文                              | 内容                       |
|-----------------------------------+----------------------------|
| [Model].objects.all()             | 全件取得                   |
| [Model].objects.filter(**kwargs)  | 条件指定（イコール）       |
| [Model].objects.exclude(**kwargs) | 条件指定（ノットイコール） |
|-----------------------------------+----------------------------|

【データ並び順】
|-----------------------------------------------+----------------------------+--------------------------------------|
| 構文                                          | 内容                       | 備考                                 |
|-----------------------------------------------+----------------------------+--------------------------------------|
| [Model].objects.order_by(*fields)             | 昇順                       |                                      |
| [Model].objects.order_by('-field1', 'field2') | field1は降順、field2は昇順 | -が降順を示す                        |
| [Model].objects.order_by('?')                 | ランダム                   | 使用しているDBによっては時間がかかる |
| [Model].objects.reverse()                     | 逆順                       |                                      |
|-----------------------------------------------+----------------------------+--------------------------------------|
