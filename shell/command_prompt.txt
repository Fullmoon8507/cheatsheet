【ファイル・フォルダの一覧情報】
|----------+------------+----------------------------------------------------------------|
| コマンド | オプション | 内容                                                           |
|----------+------------+----------------------------------------------------------------|
| dir      | -          | カーレントディレクトリのファイル名とフォルダ名をリスト表示     |
|----------+------------+----------------------------------------------------------------|
|          | /A:属性    | 指定した属性のファイルのみ表示                                 |
|          |            | 　D:ディレクトリ                                               |
|          |            | 　R: 読み取り専用                                              |
|          |            | 　H: 隠しファイル                                              |
|          |            | 　A: アーカイブ                                                |
|          |            | 　-: その属性以外                                              |
|----------+------------+----------------------------------------------------------------|
|          | /b         | ファイル名またはディレクトリ名のみ表示                         |
|          | /s         | サブディレクトリも表示する                                     |
|----------+------------+----------------------------------------------------------------|
| tree     | -          | カーレントディレクトリ以下のディレクトリ構造をツリー形式で表示 |
|----------+------------+----------------------------------------------------------------|
|          | /f         | ファイル名も含めてツリー形式で表示                             |
|          | /a         | ツリー構造をASCII文字で表示                                    |
|----------+------------+----------------------------------------------------------------|

【データ送受信】
|------------+-------------------------+--------------------------------------------+--------------------------------------|
| コマンド   | オプション              | 内容                                       | サンプル                             |
|------------+-------------------------+--------------------------------------------+--------------------------------------|
| curl [URL] | -                       | [URL]の内容を取得する。                    | curl http://localhost/               |
|            |                         |                                            | curl http://localhost:8080/          |
|            |                         |                                            | curl ftp://XXX/                      |
|------------+-------------------------+--------------------------------------------+--------------------------------------|
|            | -o {出力先ファイル}     | 取得した内容を{出力先ファイル}へ出力する。 | curl http://localhost/ -o xxx.txt    |
|            | -X {リクエストメソッド} | 指定したHTTPメソッドで取得する。           | curl -X GET http://httpbin.org/get   |
|            |                         |                                            | curl -X POST http://httpbin.org/post |
|            | -I                      | レスポンスヘッダのみを表示する。           | curl -I https://httpbin.org/         |
|------------+-------------------------+--------------------------------------------+--------------------------------------|
