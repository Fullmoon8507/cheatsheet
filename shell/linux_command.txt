****************************************************************************************************
* ネットワーク関連
****************************************************************************************************
【データ送受信】
|------------+-------------------------+--------------------------------------------------+--------------------------------------|
| コマンド   | オプション              | 内容                                             | サンプル                             |
|------------+-------------------------+--------------------------------------------------+--------------------------------------|
| curl [URL] | -                       | [URL]の内容を取得する。                          | curl http://localhost/               |
|            |                         |                                                  | curl http://localhost:8080/          |
|            |                         |                                                  | curl ftp://XXX/                      |
|------------+-------------------------+--------------------------------------------------+--------------------------------------|
|            | -o {出力先ファイル}     | 取得した内容を{出力先ファイル}へ出力する。       | curl http://localhost/ -o xxx.txt    |
|            | -X {リクエストメソッド} | 指定したHTTPメソッドで取得する。                 | curl -X GET http://httpbin.org/get   |
|            |                         |                                                  | curl -X POST http://httpbin.org/post |
|            | -I                      | レスポンスヘッダのみを表示する。                 | curl -I https://httpbin.org/         |
|------------+-------------------------+--------------------------------------------------+--------------------------------------|
| wget [URL] | -                       | [URL]の内容をダウンロードする。                  | wget http://localhost/               |
|            | -r                      | ディレクトリ配下を再帰的に全てダウンロードする。 | wget -r http://localhost/            |
|            | -r -I {階層数}          | 階層数指定して再帰的にダウンロードする。         | wget -r -I 3http://localhost/        |
|------------+-------------------------+--------------------------------------------------+--------------------------------------|

【ネットワークインターフェースの確認】
|----------+------------+------------------------------------------------|
| コマンド | オプション | 内容                                           |
|----------+------------+------------------------------------------------|
| ifconfig | -          | 現在のネットワークインターフェースの設定を表示 |
|          | -a         | 全てのネットワークインターフェースの設定を表示 |
|          | -s         | 通信状態だけを簡潔に表示                       |
|----------+------------+------------------------------------------------|

【接続確認】
|------------+-------------+-----------------------------------+-----------------------------------------|
| コマンド   | オプション  | 内容                              | サンプル                                |
|------------+-------------+-----------------------------------+-----------------------------------------|
| ping [URL] | -           | [URL]に対して検査用のパケット送信 | ping 192.168.2.1、ping www.google.co.jp |
|------------+-------------+-----------------------------------+-----------------------------------------|
| ping [URL] | -c [count]  | [count]の回数分、パケット送信     | ping -c 5 www.google.co.jp              |
| ping [URL] | -w [second] | [second]秒間、パケット送信        | ping -w 10 www.google.co.jp             |
|------------+-------------+-----------------------------------+-----------------------------------------|

【ネットワークの経路確認】
|------------------+------------+------------------------+-----------------------------|
| コマンド         | オプション | 内容                   | サンプル                    |
|------------------+------------+------------------------+-----------------------------|
| traceroute [url] | -          | 接続先までの経路を表示 | traceroute www.google.co.jp |
|------------------+------------+------------------------+-----------------------------|

****************************************************************************************************
* Archlinux関連
****************************************************************************************************
【pacman】
|----------------------------------------------+------------------------------------------------|
| コマンド                                     | 内容                                           |
|----------------------------------------------+------------------------------------------------|
| pacman -Syy                                  | リポジトリとの同期                             |
| pacman -Syyu                                 | リポジトリとの同期＋アップデート               |
|----------------------------------------------+------------------------------------------------|
| pacman -Ss パッケージ名                      | リポジトリのソフトウェア検索                   |
| pacman -Qs パッケージ名                      | ローカルにインストールされたのソフトウェア検索 |
| pacman -Qi パッケージ名                      |                                                |
| pacman -Qii パッケージ名                     |                                                |
|----------------------------------------------+------------------------------------------------|
| sudo pacman -S パッケージ名                  | パッケージのインストール                       |
| sudo pacman -Sw パッケージ名                 | インストールせずダウンロードのみ               |
| sudo pacman -U パッケージのパス パッケージ名 | ローカルにあるパッケージからインストール       |
|----------------------------------------------+------------------------------------------------|
| sudo pacman -R パッケージ名                  | パッケージの削除                               |
| sudo pacman -Rs パッケージ名                 | 依存するパッケージも削除                       |
| sudo pacman -Rns                             | バックアップファイルを削除                     |
|----------------------------------------------+------------------------------------------------|
| sudo pacman -Sc                              | キャッシュのクリーンアップ                     |
| sudo pacman -Scc                             | 完全削除                                       |
|----------------------------------------------+------------------------------------------------|
