【ctags】
●フォーマット
|-----------------+-----------------------------------------------------------------|
| 種類            | フォーマット                                                    |
|-----------------+-----------------------------------------------------------------|
| ctags           | {tagname} {TAB} {tagfile} {TAB} {tagaddress}                    |
| Exuberant ctags | {tagname} {TAB} {tagfile} {TAB} {tagaddress} {term} {field} ... |
|-----------------+-----------------------------------------------------------------|
	
●各フィールド
|--------------+-------------------------------------------------------------------------|
| フィールド名 | 説明                                                                    |
|--------------+-------------------------------------------------------------------------|
| {tagname}    | 識別子。TabとSpaceは含めることができない。                              |
| {TAB}        | Tab。                                                                   |
| {tagfile}    | {tagname}が定義されているファイル名。相対パス、絶対パスどちらでも良い。 |
|              | Tabは含めることができない。                                             |
| {tagaddress} | カーソルをタグ上に移動するExコマンド。どんなExコマンドでも使用可能。    |
|--------------+-------------------------------------------------------------------------|
| {term}       | ;" セミコロンとダブルクォートの2文字。                                  |
| {field}      | 任意のフィールドのリスト。各フィールドは次の書式を持つ。                |
|              | <Tab>{fieldname}:{value}                                                |
|              | 　{fieldname}はフィールドの識別子であり、アルファベットのみ使用可能。   |
|              | 　{value}は任意の文字列。Tabは使用できない。                            |
|--------------+-------------------------------------------------------------------------|
