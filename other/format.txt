****************************************************************************************************
* ctags
****************************************************************************************************
【フォーマット】
|-----------------+-----------------------------------------------------------------|
| 種類            | フォーマット                                                    |
|-----------------+-----------------------------------------------------------------|
| ctags           | {tagname} {TAB} {tagfile} {TAB} {tagaddress}                    |
| Exuberant ctags | {tagname} {TAB} {tagfile} {TAB} {tagaddress} {term} {field} ... |
|-----------------+-----------------------------------------------------------------|
	
【各フィールド】
|--------------+-------------------------------------------------------------------------|
| フィールド名 | 説明                                                                    |
|--------------+-------------------------------------------------------------------------|
| {tagname}    | 識別子。TabとSpaceは含めることができない。                              |
| {TAB}        | Tab。                                                                   |
| {tagfile}    | {tagname}が定義されているファイル名。相対パス、絶対パスどちらでも良い。 |
|              | Tabは含めることができない。                                             |
| {tagaddress} | カーソルをタグ上に移動するExコマンド。どんなExコマンドでも使用可能。    |
|--------------+-------------------------------------------------------------------------|
| {term}       | ;" セミコロンとダブルクォートの2文字。                                  |
| {field}      | 任意のフィールドのリスト。各フィールドは次の書式を持つ。                |
|              | <Tab>{fieldname}:{value}                                                |
|              | 　{fieldname}はフィールドの識別子であり、アルファベットのみ使用可能。   |
|              | 　{value}は任意の文字列。Tabは使用できない。                            |
|--------------+-------------------------------------------------------------------------|

****************************************************************************************************
* HTTP/1.1
****************************************************************************************************
【HTTPリクエスト】
|--------------------+------------------------------------------------------------------------|
| 名前               | 内容                                                                   |
|--------------------+------------------------------------------------------------------------|
| リクエストライン   | 「{method} /{Request URL} {HTTP_Version}」のフォーマットで設定される。 |
| ヘッダーフィールド | クライアントに関する情報が設定される。                                 |
| メッセージボディ   | GET時は空で、POST時は送信データが設定される。                          |
|--------------------+------------------------------------------------------------------------|

【HTTPレスポンス】
|--------------------+---------------------------------------------------------------------------|
| 名前               | 内容                                                                      |
|--------------------+---------------------------------------------------------------------------|
| ステータスライン   | 「{HTTP_Version} {Status_Code} {Status_Name」のフォーマットで設定される。 |
| ヘッダーフィールド | サーバーに関する情報が設定される。                                        |
| メッセージボディ   | HTMLなどリクエストした内容が設定される。                                  |
|--------------------+---------------------------------------------------------------------------|

【HTTP(主にヘッダー)の確認方法】
|----------+-----------------------------------------------------------------------------|
| ツール名 | 方法                                                                        |
|----------+-----------------------------------------------------------------------------|
| Curl     | 「Curl --verbose {URL}」のように--verboseを付けるとヘッダーも含めて見れる。 |
| Chrome   | デベロッパーツール(F12)のNetwork->Headersでヘッダーが見れる。               |
|----------+-----------------------------------------------------------------------------|
