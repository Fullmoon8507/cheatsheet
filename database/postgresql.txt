****************************************************************************************************
* CLIコマンド系
****************************************************************************************************
【データベース関連への操作】
|----------------------------------+----------------+-----------------------------------------------------------------+------------------------|
| コマンド(端末)                   | コマンド(psql) | オプション                                                      | 内容                   |
|----------------------------------+----------------+-----------------------------------------------------------------+------------------------|
| psql                             | -              | -h [ホスト名] -p [ポート番号] -d [データベース名] -U [ユーザ名] | ログインする           |
| psql                             | -              | -V                                                              | バージョン確認         |
|----------------------------------+----------------+-----------------------------------------------------------------+------------------------|
| -                                | \l             | -                                                               | データベース一覧取得   |
| -                                | \du            | -                                                               | ユーザー一覧取得       |
| -                                | \q             | -                                                               | 切断する               |
|----------------------------------+----------------+-----------------------------------------------------------------+------------------------|
| create database [データベース名] | -              | -                                                               | データベースを新規作成 |
| drop database [データベース名]   | -              | -                                                               | データベースを削除     |
|----------------------------------+----------------+-----------------------------------------------------------------+------------------------|
