****************************************************************************************************
* vim-quickrun
****************************************************************************************************
【実行】
|-----------+------------+----------------------|
| キー      | Exコマンド | 内容                 |
|-----------+------------+----------------------|
| <leader>r | :QuickRun  | 現在のバッファを実行 |
|-----------+------------+----------------------|

****************************************************************************************************
* fugitive
****************************************************************************************************
【ステータス表示】
|--------------------------+-------------------+------------------|
| Exコマンド               | 類似のGitコマンド | 内容             |
|--------------------------+-------------------+------------------|
| :G[it] or :Gstatus or :G | git status        | ステータスを表示 |
|--------------------------+-------------------+------------------|

【Gstatusウィンドウでの操作】
|-------------+------------+---------------------+------------------------------------------------|
| キー        | Exコマンド | 類似のGitコマンド   | 内容                                           |
|-------------+------------+---------------------+------------------------------------------------|
| q           |            |                     | Gstatusウィンドウを閉じる                      |
| -(ハイフン) |            | git add / git reset | 対象ファイルに対してステージング状態の切り替え |
| D           | :Gdiff     | git diff            | 対象ファイルに対してvimdiffを実施              |
| o           |            |                     | 対象ファイルに対して水平分割で開く             |
| cc          | :Gcommit   | git commit          | add済みのファイルをコミット                    |
|-------------+------------+---------------------+------------------------------------------------|

【インデックス、リポジトリへの操作】
|-------------------------+-------------------+-----------------------|
| Exコマンド              | 類似のGitコマンド | 内容                  |
|-------------------------+-------------------+-----------------------|
| :Gwrite                 | git add           | カレントファイルをadd |
| :Gwrite {file}          | git add {file}    | {file}をadd           |
|-------------------------+-------------------+-----------------------|
| :Gcommit or :Git commit | git commit        | コミットする          |
|-------------------------+-------------------+-----------------------|
| :Gpush or :Git push     | git push          | pushする              |
|-------------------------+-------------------+-----------------------|

【ログ表示】
|-----------------+-------------------+----------------------------------------------|
| Exコマンド      | 類似のGitコマンド | 内容                                         |
|-----------------+-------------------+----------------------------------------------|
| :Gclog or :Glog | git log           | quickrunにコミットログを表示                 |
| :Git blame      | git blame         | カレントファイルに対して各行の変更履歴を表示 |
|-----------------+-------------------+----------------------------------------------|

【diff操作】
|-------------------------+-----------------------------------------------------------|
| Exコマンド              | 内容                                                      |
|-------------------------+-----------------------------------------------------------|
| :Gdiff or :Gdiffsplit   | カレントファイルに対して直近のコミット内容とvimdiffを実施 |
| :Ghdiff or :Ghdiffsplit | :Gdiffを実施し、比較結果をhorizonに表示                   |
| :Gvdiff or :Gvdiffsplit | :Gdiffを実施し、比較結果をverticalに表示                  |
|-------------------------+-----------------------------------------------------------|
