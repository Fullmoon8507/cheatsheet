【強調】
 |---------------+--------------+---------------------------+----------------------|
 | 内容          | サンプル     | 出力HTML                  | 備考                 |
 |---------------+--------------+---------------------------+----------------------|
 | *{word}*      | *強調*       | <em>強調</em>             | 斜体となることが多い |
 | _{word}_      | _強調_       |                           |                      |
 |---------------+--------------+---------------------------+----------------------|
 | **{word}** 　 | **強い強調** | <strong>強い強調</strong> | 太字となることが多い |
 | __{word}__    | __強い強調__ |                           |                      |
 |---------------+--------------+---------------------------+----------------------|
