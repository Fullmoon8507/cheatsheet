****************************************************************************************************
* 読み込み
****************************************************************************************************
|------------------------+------------------------------------------------------------------------+-----------------------------------------------------------------+-----------|
| クラス                 | 内容                                                                   | サンプル                                                        | 備考      |
|------------------------+------------------------------------------------------------------------+-----------------------------------------------------------------+-----------|
| java.io.File           | ファイルおよびディレクトリのパス名の抽象表現。                         | File file = new File(fileName);                                 |           |
|------------------------+------------------------------------------------------------------------+-----------------------------------------------------------------+-----------|
| java.io.FileReader     | 文字ファイルを読み込むための簡易クラス。                               | File file = new File(fileName);                                 |           |
|                        |                                                                        | FileReader fr = new FileReader(file);                           |           |
|------------------------+------------------------------------------------------------------------+-----------------------------------------------------------------+-----------|
| java.io.BufferedReader | 文字、配列、行をバッファリングすることによって、                       | File file = new File(fileName);                                 |           |
|                        | 文字型入力ストリームからテキストを効率良く読み込みます。               | FileReader fr = new FileReader(file);                           |           |
|                        |                                                                        | BufferedReader br = new BufferedReader(fr);                     |           |
|------------------------+------------------------------------------------------------------------+-----------------------------------------------------------------+-----------|
| java.nio.file.path     | ファイル・システム内のファイルを特定するために使用可能なオブジェクト。 | Path file = Paths.get(fileName);                                | Java7以降 |
| java.nio.file.paths    | パス文字列またはURIを変換することによってPathを返す。                  |                                                                 | Java7以降 |
|------------------------+------------------------------------------------------------------------+-----------------------------------------------------------------+-----------|
| java.nio.file.files    |                                                                        | Path file = Paths.get(fileName);                                | Java7以降 |
|                        |                                                                        | // 全行読み込み                                                 |           |
|                        |                                                                        | String text = Files.readString(file);                           |           |
|                        |                                                                        | // 文字コード指定                                               |           |
|                        |                                                                        | String text = Files.readString(file, Charset.forName("UTF-8")); |           |
|                        |                                                                        | // Listで取得                                                   |           |
|                        |                                                                        | String text = Files.readAllLines(file);                         |           |
|                        |                                                                        | // byte配列で取得                                               |           |
|                        |                                                                        | byte[] bytes = Files.readAllBytes(file);                        |           |
|------------------------+------------------------------------------------------------------------+-----------------------------------------------------------------+-----------|
