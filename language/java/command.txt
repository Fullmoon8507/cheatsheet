****************************************************************************************************
* javac
****************************************************************************************************
【基本】
|-----------------------+----------------------------------------------------|
| コマンド              | 内容                                               |
|-----------------------+----------------------------------------------------|
| javac <options> <src> | options:コマンドオプション、src:Javaソースファイル |
|-----------------------+----------------------------------------------------|

【オプション】
|----------------------+----------------------------+-------------------------------------------------------|
| オプション           | 内容                       | サンプル                                              |
|----------------------+----------------------------+-------------------------------------------------------|
| -encoding 文字コード | ソースファイルの文字コード | javac -encoding utf-8 xxx.java                        |
| -sourcepath パス     | ソースファイルの保存先     | javac -sourcepath src ./src/jp/co/zzz/xxx.java        |
| -d パス              | クラスファイルの出力先     | javac -sourcepath src -d bin ./src/jp/co/zzz/xxx.java |
|----------------------+----------------------------+-------------------------------------------------------|


****************************************************************************************************
* java
****************************************************************************************************
【基本】
|------------------------------------------+----------------------------------------|
| コマンド                                 | 内容                                   |
|------------------------------------------+----------------------------------------|
| javac [options] <src> [args]             | 単一のソースファイル・プログラムを実行 |
| javac [options] -jar <jarfile> [args...] | jarファイルを実行                      |
|------------------------------------------+----------------------------------------|

【オプション】
|-------------------+---------------------------------------------------+----------------------------|
| オプション        | 内容                                              | サンプル                   |
|-------------------+---------------------------------------------------+----------------------------|
| -classpath or -cp | ディレクトリおよびzip/jarファイルのクラス検索パス | java -cp bin jp.co.zzz.xxx |
|-------------------+---------------------------------------------------+----------------------------|

****************************************************************************************************
* update-alternatives
****************************************************************************************************
【基本】
|------------------------------------+---------------------|
| コマンド                           | 内容                |
|------------------------------------+---------------------|
| update-alternatives --config javac | 使用するjavacを選択 |
| update-alternatives --config java  | 使用するjavaを選択  |
|------------------------------------+---------------------|
