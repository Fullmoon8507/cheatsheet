【配置方法】
|----------+----------------------------|
| 配置方法 | 内容                       |
|----------+----------------------------|
| pack     | 横or縦に１次元的に配列する |
| grid     | ２次元的に配置する         |
| place    | 配置を直接指定する         |
|----------+----------------------------|

【packのオプション】
|------------+--------------------------------------+----------------+------------------------------|
| オプション | 内容                                 | 値             | 設定内容                     |
|------------+--------------------------------------+----------------+------------------------------|
| side       | 方向の指定                           | tkinter.TOP    | 上から順に（デフォルト）     |
|            |                                      | tkinter.BOTTOM | 下から順に                   |
|            |                                      | tkinter.LEFT   | 左から順に                   |
|            |                                      | tkinter.RIGHT  | 右から順に                   |
|------------+--------------------------------------+----------------+------------------------------|
| expand     | ウィジェッドを配置するスペースの拡大 | True           | 拡大                         |
|------------+--------------------------------------+----------------+------------------------------|
| anchor     | どの方向に寄せるか                   | tkinter.N      | 上方向                       |
|            |                                      | tkinter.S      | 下方向                       |
|            |                                      | tkinter.W      | 左方向                       |
|            |                                      | tkinter.E      | 上方向                       |
|            |                                      | tkinter.NW     | 左上方向                     |
|            |                                      | tkinter.NE     | 右上方向                     |
|            |                                      | tkinter.SW     | 左下方向                     |
|            |                                      | tkinter.SE     | 右下方向                     |
|            |                                      | tkinter.CENTER | 中央                         |
|------------+--------------------------------------+----------------+------------------------------|
| fill       | ウィジェッドの引き伸ばし             | tkinter.NONE   | 引き伸ばししない             |
|            |                                      | tkinter.X      | 横方向に対してのみ引き伸ばす |
|            |                                      | tkinter.Y      | 縦方向に対してのみ引き伸ばす |
|            |                                      | tkinter.BOTH   | 縦横両方向に対して引き伸ばす |
|------------+--------------------------------------+----------------+------------------------------|
| padx       | ウィジェッドの外側の空白サイズを設定 | (値)           | (値)pxの空白が付加           |
| pady       |                                      |                |                              |
|------------+--------------------------------------+----------------+------------------------------|
| ipadx      | ウィジェッドの内側の空白サイズを設定 | (値)           | (値)pxの空白が付加           |
| ipady      |                                      |                |                              |
|------------+--------------------------------------+----------------+------------------------------|
