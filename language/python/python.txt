****************************************************************************************************
* 基本
****************************************************************************************************
【可変長引数】
|----------------+-------------------------+-----------------------------------------------------+-----------------------------------------------|
| 種類           | 構文                    | 例                                                  | 備考                                          |
|----------------+-------------------------+-----------------------------------------------------+-----------------------------------------------|
| タプル(*args)  | def function(*args):    | def func1(*args):                                   | 「*(タプル)」でタプル自体を引数として渡せる。 |
|                | 　[process]             | 　print(args)                                       |                                               |
|                |                         |                                                     |                                               |
|                |                         | func1("aaa", "bbb", "ccc") または                   |                                               |
|                |                         | func1(*("aaa", "bbb", "ccc"))                       |                                               |
|----------------+-------------------------+-----------------------------------------------------+-----------------------------------------------|
| 辞書(**kwargs) | def function(**kwargs): | def func1(**kwargs):                                | 「**{辞書}」で辞書自体を引数として渡せる。    |
|                | 　[process]             | 　print(kwargs)                                     |                                               |
|                |                         |                                                     |                                               |
|                |                         | func1(aaa="aaa", bbb="bbb", ccc="ccc") または       |                                               |
|                |                         | func1(**{"aaa": "aaa", "bbb": "bbb", "ccc": "ccc"}) |                                               |
|----------------+-------------------------+-----------------------------------------------------+-----------------------------------------------|

****************************************************************************************************
* クラス関連 
****************************************************************************************************
【特殊メソッド】
|--------------------------------------+-----------------------------+------------------------------------------------------------------------------------------|
| 種類                                 | 構文                        | 備考                                                                                     |
|--------------------------------------+-----------------------------+------------------------------------------------------------------------------------------|
| インスタンス生成                     | object.__new__(cls[,...])   | 静的メソッド                                                                             |
| インスタンス生成後（コンストラクタ） | object.__init__(self[,...]) | __new__()後に呼び出される。                                                              |
| インスタンス破棄時（デストラクタ）   | object.__del__(self)        |                                                                                          |
|--------------------------------------+-----------------------------+------------------------------------------------------------------------------------------|
| オブジェクト情報の生成               | object.__repr__(self)       | 戻り値は文字列オブジェクトでなければならない。                                           |
|                                      |                             | __rept__が定義してあり__str__が定義していない場合、print()などでは__rept__が使用される。 |
|--------------------------------------+-----------------------------+------------------------------------------------------------------------------------------|
| 表示情報の生成                       | object.__str__(self)        | 戻り値はstringオブジェクトでなければならない。                                           |
|                                      |                             | str(object)とformat()、print()によって呼ばれる。                                         |
|--------------------------------------+-----------------------------+------------------------------------------------------------------------------------------|

****************************************************************************************************
* その他 
****************************************************************************************************
【組み込み関数】
|----------------------------------+---------------------------+-----------------------------------------------+----------------------------------------------------------------------------|
| 種類                             | 構文                      | 例                                            | 備考                                                                       |
|----------------------------------+---------------------------+-----------------------------------------------+----------------------------------------------------------------------------|
| シンボル一覧(グローバル)取得     | globals()                 |                                               | 辞書を返す。                                                               |
| シンボル一覧(ローカル)取得       | locals()                  |                                               | 辞書を返す。                                                               |
|----------------------------------+---------------------------+-----------------------------------------------+----------------------------------------------------------------------------|
| 各iterableに対してfunctionを実行 | map(function, *iterables) | x = list(map(lambda a: a**2, range(5)))       |                                                                            |
|                                  |                           | 　-> [0, 1, 4, 9, 16]                         |                                                                            |
|----------------------------------+---------------------------+-----------------------------------------------+----------------------------------------------------------------------------|
| 複数のiterableから取得           | zip(*iterables)           | x = [i for i in range(5)]                     |                                                                            |
|                                  |                           | y = [i*2 for i in range(5)]                   | 　                                                                         |
|                                  |                           | list(zip(x, y))                               | 　                                                                         |
|                                  |                           | 　-> [(0, 0), (1, 2), (2, 4), (3, 6), (4, 8)] | 　                                                                         |
|----------------------------------+---------------------------+-----------------------------------------------+----------------------------------------------------------------------------|
| オブジェクトの属性取得           | dir(object)               |                                               | 引数に何も指定しない場合、現在のローカルスコープにある名前のリストを返す。 |
|                                  |                           |                                               | ※ビルトインの関数名と変数名はリストアップしない。                         |
|----------------------------------+---------------------------+-----------------------------------------------+----------------------------------------------------------------------------|
