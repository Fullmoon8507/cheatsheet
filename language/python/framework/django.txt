****************************************************************************************************
* 前提（環境）
****************************************************************************************************
Django vertion 3.0.4

****************************************************************************************************
* 基本
****************************************************************************************************
【プロジェクト作成】
|--------------------------------------------+------------+--------------------------|
| コマンド                                   | オプション | 内容                     |
|--------------------------------------------+------------+--------------------------|
| django-admin startproject [プロジェクト名] | -          | djangoプロジェクトを作成 |
|--------------------------------------------+------------+--------------------------|

【実行】
|---------------------+-------------------------+-------------------------------|
| コマンド            | オプション              | 内容                          |
|---------------------+-------------------------+-------------------------------|
| manage.py runserver | -                       | http://127.0.0.1:8000/で起動  |
|                     | [IPアドレス]:[Port番号] | [IPアドレス]:[Port番号]で起動 |
|---------------------+-------------------------+-------------------------------|

【アプリケーション作成】
|------------------------------------------------+------------+------------------------|
| コマンド                                       | オプション | 内容                   |
|------------------------------------------------+------------+------------------------|
| python manage.py startapp [アプリケーション名] | -          | アプリケーションを作成 |
|------------------------------------------------+------------+------------------------|

****************************************************************************************************
* Model（データベース）関連
****************************************************************************************************
【マイグレーション】
|---------------------------------+------------+--------------------------------|
| コマンド                        | オプション | 内容                           |
|---------------------------------+------------+--------------------------------|
| python manage.py makemigrations | -          | マイグレーションファイルの作成 |
| python manage.py migrate        | -          | マイグレーション               |
|---------------------------------+------------+--------------------------------|

【CRUDの基本】
- モデルクラス.モデルマネージャ.メソッド()
	- モデルクラス：django.db.models.Model継承クラス。
	- モデルマネージャ：デフォルトはobjectsという名前。
	- メソッド：SQLを実行する各メソッド。

【データ取得】
|----------------------------------+----------------------|
| 構文                             | 内容                 |
|----------------------------------+----------------------|
| [Model].objects.all()            | 全件取得             |
| [Model].objects.filter(**kwargs) | 条件指定（イコール） |
|----------------------------------+----------------------|
