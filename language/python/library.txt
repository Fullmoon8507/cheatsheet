****************************************************************************************************
* psycopg2
****************************************************************************************************
【前提】
psycopg2: 2.8.5

【サーバ接続】
|-----------------+-----------------------------------------+---------------------------------------------------------------------|
| ライブラリ      | 構文                                    | 内容                                                                |
|-----------------+-----------------------------------------+---------------------------------------------------------------------|
| import psycopg2 | conn = psycopg2.connect({database_url}) | {database_url}へ接続する。                                          |
|                 |                                         | {database_url}のフォーマットは以下の通り。                          |
|                 |                                         | ・「postgres://{username}:{password}@{hostname}:{port}/{database}」 |
|-----------------+-----------------------------------------+---------------------------------------------------------------------|

【カーソル】
|---------------------+----------------|
| 構文                | 内容           |
|---------------------+----------------|
| cur = conn.cursor() | カーソルを取得 |
|---------------------+----------------|

【SQL実行】
|--------------------+-----------------|
| 構文               | 内容            |
|--------------------+-----------------|
| cur.execute({SQL}) | {SQL}を実行する |
|--------------------+-----------------|

【クローズ】
|-------------+--------------------|
| 構文        | 内容               |
|-------------+--------------------|
| cur.close() | カーソルをクローズ |
| conn.cose() | 接続をクローズ     |
|-------------+--------------------|
