【用語】
|------+------|
| 用語 | 内容 |
|------+------|

【初期化】
|---------------------------------+------------+------------------------------------------------------------|
| コマンド                        | オプション | 内容                                                       |
|---------------------------------+------------+------------------------------------------------------------|
| git init                        | -          | カーレントディレクトリにリポジトリを新規作成または初期化   |
|---------------------------------+------------+------------------------------------------------------------|
| git clone [from repository url] | -          | カーレントディレクトリに指定したリポジトリのクローンを作成 |
|---------------------------------+------------+------------------------------------------------------------|

【設定】
|------------+----------------------------------------------------+----------------------------|
| コマンド   | オプション                                         | 内容                       |
|------------+----------------------------------------------------+----------------------------|
| git config | --list                                             | 全ての設定を確認           |
|            | --system --list                                    | システム全体の設定を確認   |
|            | --global --list                                    | ユーザ全体の設定を確認     |
|            | --local --list                                     | 対象リポジトリの設定を確認 |
|------------+----------------------------------------------------+----------------------------|
| git config | --[system or global or local] user.name (UserName) | ユーザ名を設定             |
|            | --[system or global or local] user.email (Email)   | メールアドレスを設定       |
|------------+----------------------------------------------------+----------------------------|
